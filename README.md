# game
play a game

Борьба с невзгодами
Roguelike-шутер с видом сверху. Сражайтесь в случайно генерируемых подземельях, собирайте монеты, покупайте оружие и улучшения, побеждайте боссов и становитесь сильнее с каждым этажом.

---

## 🎮 Управление

| Клавиша | Действие |
|---|---|
| W / A / S / D | Движение персонажа |
| Левая кнопка мыши | Атака / Выстрел |
| Правая кнопка мыши | Зарядка лазера (для дальнего оружия) |
| TAB | Открыть магазин |
| F | Подобрать предмет / Купить оружие |
| 1 / 2 | Переключить оружие |
| Пробел | Активировать навык |
| ESC / P | Пауза |
| F11 | Полноэкранный режим |

---

## 🚀 Запуск игры

Установка зависимостей:

pip install pygame

Запуск:

python 2026.04.17lls.py

---

## 🧬 Игровая механика

- Генерация карты: каждый этаж создаётся случайным образом
- Боевая система: ближний бой, дальние атаки, гранаты, огнемёт
- Магазин: нажмите TAB, чтобы купить улучшения
- Таланты: выбирайте один из трёх талантов каждые 3 этажа
- Боссы: каждый 5-й этаж встречается босс
- Сложность: враги становятся сильнее с каждым этажом
